# D.I.G.E. - 帝江号严选精细化集成工业系统能源生产及存储管理系统设计器

一个用于计算明日方舟：终末地中热能池最优发电方案的工具。

## 功能特性

- **智能计算**：自动计算满足目标功率的最优发电方案
- **多方案对比**：同时展示最多 5 种可行方案，按复杂度、稳定性排序
- **可视化展示**：
  - 周期图表：实时显示发电功率和蓄电池电量变化
  - 流程图：清晰展示分流器连接方式和存储箱配置
- **多语言支持**：中文、English、日本語、한국어
- **响应式设计**：支持桌面端和移动端

## 技术栈

- React 18
- Vite
- Tailwind CSS v4
- Chart.js

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 使用说明

### 参数设置

1. **目标发电量**：设置你需要的发电功率（单位：w）
2. **最小蓄电量**：允许的最低蓄电池电量百分比
3. **功率浪费上限**：允许的最大溢出功率
4. **主燃料**：用于基础发电和震荡发电的燃料类型
5. **副燃料**：可选，用于震荡发电的备选燃料

### 燃料参数

| 燃料类型 | 发电功率 | 燃烧时间 |
|---------|---------|---------|
| 源矿 | 50w | 8s |
| 低容谷地电池 | 220w | 40s |
| 中容谷地电池 | 420w | 40s |
| 高容谷地电池 | 1100w | 40s |
| 低容武陵电池 | 1600w | 40s |

### 系统常量

- 基础发电功率：200w
- 传送带速度：0.5 个/秒
- 蓄电池容量：100,000 J

### 方案解读

- **基础发电**：使用满带主燃料的发电机数量
- **震荡发电**：通过分流器实现的精确功率控制
  - 2分器：一入二出，向上分流回收
  - 3分器：一入三出，向上向下分流回收
  - 所有回收线路必须接入存储箱（设为存储模式或不通电）

## 注意事项

⚠️ **重要**：分流回收线路必须使用存储箱作为缓存，存储箱需设为存储模式或不通电，切勿直接接入传送带！
